## What Was Built

A self-improving ML pipeline for Airbnb listing optimization. Every process reads learned strategy, emits signals, and updates the shared brain after each run. Two learning loops:

- **Intra-process**: each process reads its own strategy before running, emits signals after. The generator learns preferred title length from high-rated listings it has seen; the analyzer learns which amenity gaps are most correlated with low occupancy.
- **Inter-process**: `recompute_all_strategies()` runs after every batch — outcome signals improve generation, scraper signals improve delivery targeting, analyzer signals tighten the amenity alert threshold.

### Files

| File | Purpose |
|---|---|
| `scraper.py` | Apify-backed harvester. 67 fields per listing. Emits 8 ML signals per run. |
| `learning_store.py` | SQLite brain (`learning.db`). `emit()`, `get_strategy()`, `set_strategy()`, `recompute_all_strategies()`. |
| `teardown.py` | **THE PRODUCT.** One Claude call → 5 scores + 3 title variants + rewritten desc + amenity gaps + outreach hook + PDF. ~$0.01-0.03/listing. |
| `sheet_writer.py` | Google Sheets + Drive writer. 50-column schema with ML columns pre-wired. Needs OAuth token (see below). |
| `pipeline.py` | CLI orchestrator. `scrape`, `status`, `signals` subcommands. |
| `gap_analysis_pdf_generator.py` | Existing. Generates PDF report via reportlab. |
| `description_improver.py` | Existing. `improve_listing_description()`, `generate_title_suggestions()`. |
| `email_service.py` | Existing. Delivery. ALWAYS test-send to yourself first. |

## Fable 5 Strategic Evaluation

### Core finding

The pipeline is built. The only thing between zero and first deal is **finding the right prospect and sending the teardown**.

The demo listing (Finley's Fort) has 80% occupancy — wrong ICP. High-occupancy hosts don't feel pain. The pipeline needs to be pointed at hosts with:
- Occupancy 40-70% (enough pain to act, enough volume to prove ROI)
- Rating 4.3-4.7 (room to improve, not a lost cause)
- Nightly rate $150+ (meaningful revenue at stake)
- 1-5 listings (owner-operator, not a property manager with a team)
- Weak copy (title > 55 chars, generic description, missing amenity labels)

### The $0 path to 1 deal this week

**Step 1 — Find 5 real prospects (today)**

Manual: Go to Airbnb, search a specific market (Bali, Ubud, Canggu — you know these), filter for under-$200/night entire homes. Open 5 listings that look like they have weak copy. Run `python teardown.py <url>` on each. Takes ~15 minutes total including scrape time (~$0.02 Apify + ~$0.03 Claude per listing = $0.25 total).

Signals that make a good prospect:
- Title is a wall of text or starts with the property name (not a differentiator)
- Description doesn't mention the strongest amenity in the first 2 sentences
- Calendar has obvious open blocks on weekends

**Step 2 — Eyeball the teardown output (5 min per listing)**

Check the JSON at `/tmp/listingboost_teardowns/teardown_{id}.json`. Look for:
- `revenue_at_stake_90d` > $3,000 (enough to make the host care)
- `outreach_hook` that uses a specific, verifiable observation from their listing
- `score_title` < 5 (clear room to improve)

Discard listings where `occupancy_pct` > 75 or `revenue_at_stake_90d` < $1,500.

**Step 3 — Send the outreach email (same day)**

Use the `outreach_hook` from teardown output as your opener. Never fabricate numbers. The hook is already personalized to their listing data.

Template structure:
```

just the fixes. If it's useful, I'd love to show you what the full optimization looks like.

AL
ListingBoost
```

Attach the PDF from `pdf_path`.

Sign as AL. Test-send to yourself first.

**Step 4 — Follow up if no reply in 3 days**

One follow-up only. Reference the specific fix: "Did you get a chance to look at the WiFi amenity note?" Not "just following up."

- Occupancy 40-70% (visible open blocks in calendar)

**Anti-ICP (skip immediately):**
- Occupancy > 75%
- Superhost with 500+ reviews and clean listing (already optimized)
- Property management company (not owner-operator)
- Listing with professional photography and polished copy (wrong ICP)
