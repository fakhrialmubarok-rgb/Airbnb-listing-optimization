================================================================================
                         API ENDPOINTS
================================================================================

GET /
  - Serves the web interface
  - Beautiful landing page with features
  - Pricing information

GET /health
  - Health check endpoint
  - Returns: {"status": "ok"}

POST /api/improve-description
  - Input: {"description": "listing text..."}
  - Output: {"improved": "enhanced text..."}
  - Powered by: Claude 3.5 Sonnet

POST /api/generate-titles
  - Input: {"property_type": "Apartment", "key_features": "...", "location": "..."}
  - Output: {"titles": ["Title 1", "Title 2", ...]}
  - 5 unique, optimized options

POST /api/enhance-images
  - Input: multipart/form-data with images
  - Output: {"images": {filename: {"base64": "data:image/jpeg;base64,..."}}
  - Processes multiple images simultaneously

POST /api/preview
  - Combined endpoint for full preview
  - Returns improved description + titles
  - No images in this endpoint
